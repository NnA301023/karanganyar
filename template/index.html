APLIKASI ANALISIS SENTIMEN TWITTER TERHADAP WISATA KABUPATEN KARANGANYAR

Karanganyar memiliki beragam objek wisata yang dapat dikunjungi oleh wisatawan, atas dasar evaluasi pelayanan saya membuat sebuah website yang dapat menganalisa komentar pengunjung terhadap suatu objek wisata pada sosial media berupa twitter. Sistem Sentiment Analisis kami menggunakan model machine learning naive bayes sebagai model yang memprediksi suatu komentar mengandung makna positif, negatif, atau netral. Untuk menganalisis, user memasukkan hastag pada form input hastag bagian pojok kanan. Setelah memasukkan hastag, user menekan button search di dekat form input hastag.
